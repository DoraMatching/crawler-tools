# Crawler Tools

## Install yarn and sesquelize-cli

```bash
[sudo] npm install yarn sequelize-cli -g
```

## Clone project

```bash
git clone https://github.com/DoraMatching/crawler-tools.git
```

## Install dependencies

```bash
cd crawler-tools
yarn install
```

## Migration

```bash
sequelize db:migrate
```

## Start crawling data

```bash
yarn start
```
